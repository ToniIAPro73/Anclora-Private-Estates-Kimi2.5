# menu-overlay-clarity-redesign-test-plan-v1

Feature ID: ANCLORA-MENU-002

## Casos funcionales
1. Abrir menu desde header y cerrarlo con click en cerrar, overlay exterior y tecla `Escape`.
2. Verificar lista principal con grupos y flechas de avance.
3. Entrar en submenu y volver a la lista principal.
4. Navegar a secciones (`hero`, `properties`, `valuation`, `insights`, `about`, `contact`, `philosophy`, `invest`, `neighborhood`).
5. Validar `Agent Portal` redirige a URL configurada.
6. Validar `Partner Portal` y `Data Lab` abren modal informativo.

## Casos UX/visual
1. Overlay claro, limpio y sin cards.
2. Jerarquia visual legible en desktop y mobile.
3. Estados hover/focus visibles y consistentes con estilo premium.

## Casos i18n
1. Verificar textos nuevos en `es`, `en`, `de` y `fr`.
2. Verificar cambio de idioma dentro del menu sin roturas.
